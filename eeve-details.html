EeVe – Smart Event Assistant for Riyadh

EeVe is a mobile app designed to help users in Riyadh find events that match their mood, preferences, and budget—instantly. Instead of endless scrolling, users can simply chat with EeVe’s AI assistant to get personalized recommendations for nearby events like concerts, cafes, or festivals.

🛠️ Key Features

  • AI Chat Assistant: GPT-powered responses tailored to each user query.
  • Local Focus: Events filtered exclusively for Riyadh users.
  • Real-Time Matching: Pulls live event data to match mood, time, and budget.
  • Modern UI/UX: Flutter + GetX + Light & Dark Mode for elegant performance.

🧩 Technologies Used

  • Flutter + Dart for cross-platform development
  • Supabase for event data and auth
  • OpenAI GPT-3.5 for chat response
  • Figma for collaborative UI/UX design
  • Flutter packages: flutter_credit_card, provider, image_picker, dart_openai, etc.
